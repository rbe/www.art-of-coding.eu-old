< Was wir leisten>

Java ist eine <objektorientierte/> Sprache, die grenzenlose Programmierung möglich macht. Weil sie plattformunabhängig ist. Weil sie vielseitig ist. Weil sie offen ist.

Java zeichnet sich durch einen <portablen Code/> aus, der nach der Maxime <write once, run anywhere/> sowohl für UNIXe, Windows und Mac OS X genutzt werden kann. Diese Plattformunabhängigkeit macht die Arbeit mit Java effizient und universell.

Java verfügt über eine sehr umfangreiche Bibliothek und ist daher für sehr viele Einsatzzwecke nutzbar. Aufgrund dieser Vielseitigkeit wird Java als <General Purpose Language/> klassifiziert.



Java zeichnet sich wie keine andere Sprache durch eine besondere Offenheit aus. Auch aus diesem Grund gehörte Java in den letzten 10 Jahren immer zu den Top 3 der mitbenutzen Programmiersprachen der Welt; aktuell rangiert sie im TIOBE-Index auf Rang 2.

Java ist eine Plattform, mit der sich Programme schreiben lassen, die <parallelisierbar/> sind und die auf verschiedene Rechner <verteilbar/> ist. Das macht sie auch aus Performance-Gesichtspunkten attraktiv. Das mehrstufige Sicherheitskonzept stellt ein weiteres Qualitätsmerkmal von Java dar.

 

 

Art of Coding arbeitet mit Java, um Anwendungen für Unternehmen zu entwickeln, die über das Web für den Browser, den Desktop oder für mobile Geräte nutzbar sein sollen.
